My Playground

These are the tools I love experimenting with - each one opens up new possibilities for innovation. I'm drawn to platforms that let me quickly test ideas, combine different technologies, and explore what happens when you push boundaries with emerging capabilities.

Anthropic Claude 4

My primary AI partner for complex reasoning, nuanced conversations, and creative problem-solving. Perfect for experimenting with AI that truly understands context and provides thoughtful, intelligent responses.

Supabase

My favorite backend-as-a-service for quick database setups and real-time features. Makes it easy to focus on the interesting parts of an experiment.

Pinecone

Vector database that makes semantic search and RAG experiments possible. Essential for building AI that can work with your own data intelligently.

n8n

Visual automation platform where I connect different services and test workflow ideas. Love how it lets me experiment with complex integrations without heavy coding.

Microsoft Power Automate

Great for exploring automation within the Microsoft ecosystem. Useful for testing how new ideas might work in corporate environments.

Airtable

Flexible database that's perfect for organizing experiments and managing project data. The sweet spot between spreadsheets and proper databases.

PostgreSQL

Reliable database for when experiments need more robust data handling. My go-to when prototypes start getting serious about data integrity.

Cline

AI-powered coding assistant that accelerates experimentation and rapid prototyping. Perfect for turning ideas into working code quickly while focusing on innovation rather than syntax.
